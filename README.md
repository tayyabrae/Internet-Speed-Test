# Internet Speed Test Project

## Overview

This project is a simple web application to test your internet speed by calculating the download speed (in Mbps).

## Features

- Measures download speed using a test image.
- Displays results in Mbps.
- Provides a clean and responsive UI.

## Technologies Used

- **HTML** for structure.
- **CSS** for styling.
- **JavaScript** for functionality.

## How to Use

1. Open the `index.html` file in any modern browser.
2. Click the "Check" button to test your internet speed.
3. Results will be displayed on the screen.

## Folder Structure

- **index.html**: Main HTML file.
- **style/style.css**: Stylesheet for the project.
- **js/script.js**: JavaScript functionality.
- **assets/images/**: Placeholder for images.
